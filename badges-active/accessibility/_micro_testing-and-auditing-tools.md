Accessibility - Testing and Auditing [Micro Badge]
===================================================

Est. time to complete: 2-3h

Goals:
------

- You understand how to perform a manual accessibility audit and set up accessibility tooling


Intro
-----

> Web accessibility testing is a subset of usability testing where the users under consideration have disabilities that affect how they use the web. The end goal, in both usability and accessibility, is to discover how easily people can use a web site and feed that information back into improving future designs and implementations. â€“ [Accessibility Testing - W3C](https://www.w3.org/wiki/Accessibility_testing)



Suggested Assignment(s)
---------------------

Do both:

1) Add an automated accessibility testing tool to a project

2) Perform a manual accessibility audit on a site of your choosing
